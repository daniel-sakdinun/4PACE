===================================
Welcome to 4PACE's Documentation!
===================================

**Four-Quadrant Power Analysis & Computational Engine (4PACE)** is a high-precision Power System Optimization and Analysis Engine designed for modern electrical grids. Built entirely in Python, 4PACE handles everything from microgrid investment planning to dynamic transient stability on complex mesh networks.

🚀 Core Features
================
* **Ultimate Transient Stability:** OOP ODE Engine for IEEE dynamic models and CCT Binary Search.
* **Iterative Fault Engine (IBR Ready):** Asymmetrical fault solver with smart inverter current limits.
* **Capacity Expansion Planning (CEP):** Co-optimize CapEx and OpEx.
* **Preventive SCOPF:** Multiverse N-1 survivability.
